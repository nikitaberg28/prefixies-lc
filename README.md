# префиксы легендкрафт

здесь хранятся префиксы игроков, которые отображаются в боте при просмотре онлайна в инлайн боте [@LegendCraftUtils_Bot](https://t.me/LegendCraftUtils_Bot)

## как добавить себя?

### сначала создай аккаунт на GitHub
- зайди на [github](https://github.com/login) и зарегистрируйся
- далее зайди во вкладку [Issues](../../issues)
- добавь туда свой issue таким форматом:
<img width="1033" height="342" alt="image" src="https://github.com/user-attachments/assets/fffeb0c1-0395-4d03-939f-17cd643522b0" />

в пример могу привести такой:
`BaralikRogalik28: крутойхацкер`
- далее нажми кнопку `Create` чтобы создать Issue
- жди когда твой префикс добавят
 > а может и не добавят.
